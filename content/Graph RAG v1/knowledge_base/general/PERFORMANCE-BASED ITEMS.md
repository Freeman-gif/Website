# PERFORMANCE-BASED ITEMS

## Description

Performance-based items are test questions that require students to perform tasks rather than select answers, thus often being more costly to administer

## Related Concepts

- [[STATE TESTING]] — State testing programs also include performance-based items, which are generally more expensive to grade
