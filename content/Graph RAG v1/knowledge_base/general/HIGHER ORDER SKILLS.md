# HIGHER ORDER SKILLS

## Description

Higher order skills refer to advanced cognitive skills that students are expected to develop for employment and are a focus in education assessments.

## Related Concepts

- [[NAEP]] — NAEP tests aim to evaluate students' mastery of higher order skills necessary for success in modern society.
- [[TIMSS]] — The TIMSS tests focus on assessing higher order skills to prepare students for future employment in society.
