# ADAPTIVE TESTING

## Description

Adaptive testing refers to a method of assessment that adjusts to a student's ability level and has been implemented for mathematics but not yet recommended for science.

## Related Concepts

- [[MATHEMATICS]] — Adaptive testing has been implemented in large-scale systems specifically for mathematics
- [[SCIENCE]] — Adaptive testing poses challenges for science due to the wide range of topics involved, making it difficult to apply across different areas
