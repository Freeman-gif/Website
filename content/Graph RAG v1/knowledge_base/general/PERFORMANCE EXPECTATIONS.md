# PERFORMANCE EXPECTATIONS

## Description

Performance expectations are defined as the goals for what students should be able to do using their understanding of science according to the NGSS.

## Related Concepts

- [[ASSESSMENTS]] — The assessments are meant to evaluate whether students meet the performance expectations set by the NGSS
- [[NGSS]] — The NGSS outlines performance expectations that represent a shift in focus from knowledge to practical scientific applications
