# STANFORD NGSS ASSESSMENT PROJECT

## Description

The Stanford NGSS Assessment Project is an initiative dedicated to creating a comprehensive system of assessment aligned with the Next Generation Science Standards (NGSS) in California. This project receives funding from the S.D. Bechtel Jr. Foundation, which supports its efforts to improve educational practices in science.

The project's primary focus is to develop and refine assessment tasks to ensure they meet the criteria and standards set by the NGSS. This involves a thorough review and evaluation process where assessment tasks are scrutinized for their alignment with the core principles and performance expectations of the standards. The aim is not only to create reliable assessments but also to enhance the overall quality of science education across the state.

Through these efforts, the Stanford NGSS Assessment Project seeks to provide educators with effective tools and resources that can be used to measure students' understanding and application of scientific concepts. By aligning assessments with NGSS standards, the project aims to promote deeper learning and student engagement in science by fostering a curriculum that emphasizes inquiry-based and hands-on learning experiences.
