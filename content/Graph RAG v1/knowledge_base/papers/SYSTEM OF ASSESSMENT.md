# SYSTEM OF ASSESSMENT

## Description

The System of Assessment refers to the framework proposed for evaluating and improving science education in California.

## Related Concepts

- [[CALIFORNIA SCIENCE ASSESSMENT]] — The System of Assessment is the overarching framework under which the California Science Assessment operates.
- [[NEXT GENERATION SCIENCE STANDARDS]] — The System of Assessment is designed to evaluate student performance based on the Next Generation Science Standards.
- [[ASSESSMENT SUPPORTS]] — Assessment supports are part of the System of Assessment to help implement NGSS effectively.
- [[PERFORMANCE TASKS]] — Performance tasks are integral to the overall System of Assessment proposed in California to align with student learning outcomes.
