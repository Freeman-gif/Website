# ACHIEVEMENT LEVELS

## Description

Standards set by NAEP to categorize student performance outcomes based on their assessment scores.

## Related Concepts

- [[NATIONAL ASSESSMENT GOVERNING BOARD]] — The Governing Board develops and sets the achievement levels based on the assessment results
- [[NATION’S REPORT CARD]] — Achievement levels are reported in the Nation’s Report Card to help the public and policymakers interpret student performance results
